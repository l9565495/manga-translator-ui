# v2.1.3 更新日志

发布日期：2026-02-26

## ✨ 新增

- 新增“膨胀不超过气泡蒙版”：
  - 限制蒙版膨胀范围不超出气泡区域，防止气泡边框在修复时被误擦除。

## 🐛 修复

- 修复自动断句在“单区域（OCR lines=1）+ 无 `[BR]`”场景下仍被强制插入 `[BR]` 的问题：
  - `solve_no_br_layout(...)` 读取当前区域上下文后，单区域时不再主动断句。
  - 适用于智能气泡相关流程（包含 `smart_scaling` / `balloon_fill` 走到 no-br 自动布局的路径）。
  - 保持多区域场景原有自动断句行为不变。
